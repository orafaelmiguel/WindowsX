TCP/IP Optimization

The TCP/IP optimization script automatically adjusts network settings for better performance:

# Adjust TCP window size
Set-NetTCPSetting -SettingName InternetCustom -InitialCongestionWindow 10
Set-NetTCPSetting -SettingName InternetCustom -CongestionProvider CTCP

# Optimize receive buffer
Set-NetTCPSetting -SettingName InternetCustom -ReceiveWindowAutoTuningLevel Normal

These settings are automatically applied when you run the network optimization script.
